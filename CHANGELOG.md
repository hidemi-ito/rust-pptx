# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-25

### Added
- Initial release
- Create and open PowerPoint (.pptx) files
- Slide management (add, delete, reorder, layouts)
- Shapes: AutoShapes, TextBoxes, Pictures, Tables, Connectors, Groups, Freeform
- Text formatting: fonts, colors, bullets, paragraphs, hyperlinks
- Charts: bar, line, pie, scatter, bubble, stock, surface, radar, area
- Drawing ML: fills, lines, shadows, 3D effects, colors
- Core document properties
- Slide transitions and animations
- Comments and sections
- SmartArt reading support
- OLE object support
- Embedded fonts extraction
- VBA macro detection
