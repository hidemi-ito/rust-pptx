# Contributing to rust-pptx

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/rust-pptx.git`
3. Create a branch: `git checkout -b my-feature`
4. Make your changes
5. Run checks: `cargo fmt && cargo clippy -- -D warnings && cargo test`
6. Commit and push your changes
7. Open a Pull Request

## Development

### Prerequisites
- Rust 1.85 or later

### Running Tests
```bash
cargo test
```

### Code Style
- Run `cargo fmt` before committing
- Ensure `cargo clippy -- -D warnings` passes with no warnings
- Add tests for new functionality
