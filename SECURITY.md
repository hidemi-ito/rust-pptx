## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public GitHub issue.
2. Use [GitHub's private security advisory](https://github.com/hidemi-ito/rust-pptx/security/advisories/new) to report the vulnerability.
3. You will receive a response within 7 days acknowledging the report.

Thank you for helping keep this project safe.
